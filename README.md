## Issues Encountered

### Claude Code installation

Issue:
Claude Code was not directly available through the Cursor interface.

Solution:
Installed Claude Code through npm and configured it manually.

### Node.js dependency

Issue:
npm commands required Node.js.

Solution:
Installed the latest LTS version of Node.js.

### PowerShell execution policy

Issue:
Windows initially blocked npm script execution.

Solution:
Updated the execution policy using:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

### Git installation

Issue:
The git command was not recognized.

Solution:
Installed Git for Windows and restarted Cursor.
